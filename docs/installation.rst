Installation
************

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install steempy

Manual installation:

::

    $ git clone https://github.com/holgern/steempy/
    $ cd steempy
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
